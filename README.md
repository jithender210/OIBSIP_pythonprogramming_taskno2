# 🧮🧮BMI Calculator with GUI (Tkinter)

A user-friendly **Body Mass Index (BMI) Calculator** built using **Python Tkinter**.  
This application allows users to calculate BMI, store records, view history, and analyze BMI trends with graphs.
## 🚀 Features

- ✅ Simple and intuitive GUI using Tkinter  
- ✅ BMI calculation (kg & cm inputs)  
- ✅ BMI category classification:
  - Underweight
  - Normal
  - Overweight
  - Obese  
- ✅ Store multiple user records (SQLite database)  
- ✅ View BMI history  
- ✅ Visualize BMI trends using graphs (Matplotlib)  

## 🛠️ Technologies Used

- Python 3.x  
- Tkinter (GUI)  
- SQLite3 (Database)  
- Matplotlib (Graph visualization)
